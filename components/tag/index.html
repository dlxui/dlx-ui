Tag 标签 ​

用于标记和分类。

基本用法 ​

:::demo

vue
<template>
  <div>
    <Tag>Tag 1</Tag>
    <Tag>Tag 2</Tag>
    <Tag>Tag 3</Tag>
  </div>
</template>

:::

不同颜色 ​

:::demo

vue
<template>
  <div>
    <Tag color="red">Tag 1</Tag>
    <Tag color="blue">Tag 2</Tag>
    <Tag color="green">Tag 3</Tag>
  </div>
</template>

:::

不同尺寸 ​

:::demo

vue
<template>
  <div>
    <Tag size="small">Tag 1</Tag>
    <Tag size="medium">Tag 2</Tag>
    <Tag size="large">Tag 3</Tag>
  </div>
</template>

:::

带关闭按钮的标签 ​

:::demo

vue
<template>
  <div>
    <Tag closable>Tag 1</Tag>
    <Tag closable>Tag 2</Tag>
    <Tag closable>Tag 3</Tag>
  </div>
</template>

:::
